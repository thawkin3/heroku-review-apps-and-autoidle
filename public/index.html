Heroku Review Apps and AutoIdle

This project utilizes Heroku to create review apps for pull requests in GitHub. It also uses AutoIdle to automatically put review apps to sleep when not in use, saving you money.

Getting Started on Heroku with Node.js Source on GitHub

High Level Step-by-Step Process

  1. Create a Heroku app
  2. Create a Heroku pipeline
  3. Enable review apps for the pipeline
  4. Add your Heroku app to the pipeline in the Production section
  5. Create a new pull request on the repo
  6. Notice that the review app is created and a link to it appears on the pull request
  7. View the review app to see your pull request's changes
  8. Merge the pull request
  9. Notice that the review app is deleted
